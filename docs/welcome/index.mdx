# Welcome to the Wamule Operations Platform

**Wamule Development** uses the **Wamule Operations Platform** to manage its housing projects, available lots, buyer inquiries, applications, reservations, customers, contracts, payments, collections, and post-sale activities.

The platform brings the company’s sales, administrative, and financial workflows into one central system. It helps staff maintain accurate records, follow each buyer’s progress, manage lot availability, record payments, identify overdue accounts, and coordinate the work required after a sale.

This guide explains the platform’s features, the responsibilities of each user, and the recommended workflows for managing Wamule Development’s daily operations.

<Boundary>
The Wamule Operations Platform is a custom workflow system used by Wamule Development. It is not currently presented as a commercial software subscription or public marketplace product.
</Boundary>

## How to use this guide

Use the sidebar to find a feature, workflow, or SOP. Articles state what the system does, who owns the work, the record to update, and what must be checked before moving forward. The search icon searches this guide locally in your browser.

:::tip Keep the record current
The platform is most useful when each buyer has an accurate stage, a named owner, a recorded next action, and a clear activity history.
:::
