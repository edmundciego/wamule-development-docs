# Inquiry to lead

Public visitors can submit a land application and a separate information inquiry. A submitted public application becomes an application record and, where the deployed trigger is active, creates a linked sales lead and follow-up. A public information inquiry creates a lead-oriented sales record through its dedicated server workflow.

Staff should:

1. Review the new item promptly.
2. Check possible duplicate indicators and linked records.
3. Confirm contact details, source, lot interest, and assigned staff member.
4. Create or verify a dated first-contact task.

:::caution Do not assume deployment state
Confirm application-to-lead automation and public-inquiry routing in the authorized live/test environment before using it as a guaranteed operational commitment.
:::
