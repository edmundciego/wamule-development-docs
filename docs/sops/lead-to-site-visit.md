# SOP — Lead to site visit

**Purpose:** schedule, complete, and follow up on a buyer viewing appointment.

| Field | Standard |
| --- | --- |
| Trigger | Qualified buyer requests or accepts a viewing. |
| Responsible role | Assigned sales staff. |
| Required information | Buyer, lot/project context, date/time, location, host, preparation notes. |
| Steps in platform | Create site visit from lead; set staff/date/location; confirm buyer; record final outcome and new follow-up. |
| Expected status changes | Lead `site_visit_scheduled`; visit `scheduled` → completed/no-show/cancelled/rescheduled. |
| Required documents | None unless company process requires a site-visit form. |
| Financial checks | None. |
| Follow-up deadline | Confirm before visit and record outcome the same business day. |
| Escalation conditions | Safety/location issue, unavailable lot, or buyer requests a hold/payment action. |
| Completion criteria | Visit outcome and subsequent buyer action are documented. |
| Audit/accountability evidence | Visit record, lead activity, follow-up task. |
| Common mistakes | Treating a visit as a reservation; failing to record no-show/reschedule. |

**Media:** Screenshot, video, checklist.
