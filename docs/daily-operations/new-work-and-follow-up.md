# Processing new work and buyer follow-up

1. Review new public inquiries, applications, and leads.
2. Check **Possible Duplicate** indicators before creating parallel work.
3. Confirm contact details, preferred lot, assigned owner, stage, and next action.
4. Record calls, WhatsApp, email, notes, follow-ups, and site-visit outcomes in the lead activity history.
5. Reschedule or complete overdue follow-ups; close inactive work with a documented reason.

Every active lead should have a meaningful next action unless it is intentionally inactive.
