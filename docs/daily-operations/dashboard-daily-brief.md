# Reviewing the Dashboard and Daily Brief

Use the Dashboard for operational totals and alert queues. It highlights sales movement, follow-ups, site visits, reservation/deposit readiness, collections, post-sales work, and lot counts.

Use Daily Brief to compare current priorities with the previous brief and manage open action items. Mark an action item done or dismissed only after the underlying work is complete or no longer applicable.

Do not treat an AI-generated or rule-based summary as a final decision. Open the related lead, application, customer, payment, reservation, or contract record to verify it.

**Media:** Annotated screenshots, short walkthrough video, fictional scenario.
