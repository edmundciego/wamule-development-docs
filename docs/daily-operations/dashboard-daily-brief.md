# Reviewing the Dashboard and Daily Brief

Use the Dashboard for live operational totals and source-backed alert queues. Today’s Attention identifies the affected buyer or customer, action, deadline, owner, status, and destination. Metric cards with non-zero values should open the filtered records behind the count.

Use Daily Brief to compare current priorities with the previous brief and manage open action items. Each saved source-backed item is revalidated against the live record and may be Current, Updated since brief, Resolved since brief, or Source unavailable. Mark an action item done or dismissed only after the underlying work is complete or no longer applicable.

Do not treat an AI-generated or rule-based summary as a final decision. Open the related lead, application, customer, payment, reservation, or contract record to verify it.
