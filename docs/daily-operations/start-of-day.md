# Start-of-day workflow

**Purpose:** turn current records into a prioritized, owned work list before new work begins.

1. Open the [Dashboard](../feature-guides/dashboard).
2. Review overdue follow-ups, applications, upcoming visits, reservation/deposit exceptions, collections alerts, and post-sales blockers.
3. Open the [Daily Brief](../feature-guides/daily-brief) when authorized; verify important facts in the underlying records.
4. Assign or confirm an owner and due date for each priority item.
5. Start with urgent financial, lot-conflict, application, or access exceptions.

:::note Daily Brief boundary
The Daily Brief is advisory and tracks action items. It does not approve applications, reconcile payments, reserve lots, or send email automatically.
:::
