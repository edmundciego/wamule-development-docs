# Platform glossary

| Term | Meaning |
| --- | --- |
| Application | Buyer’s submitted land application, reviewed as Pending Review, Approved, or Declined. |
| Customer | Buyer record created/linked after approval. |
| Daily Brief | Advisory daily summary and action tracking, not an automation authority. |
| Deposit readiness | Operational deposit expectation/evidence status, not a ledger payment. |
| Lead | Sales prospect and buyer-follow-up record. |
| Lot / parcel | Land inventory record with core availability status. |
| Payment request | Expected-payment record; it does not itself prove payment. |
| Post-sales | Coordinated tasks and readiness after approval/contract start. |
| Reservation | Temporary operational hold with expiry/deposit context. |
| Smart insight | Rule-based or AI-generated advisory guidance. |
