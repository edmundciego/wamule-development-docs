# Navigation overview

The protected sidebar is the primary workspace. It contains Dashboard, Daily Brief, Leads, Lots, Applications, Customers, Contracts, Payments, Collections, Reports, Audit Trail, Settings, and **Send Feedback**. The Email Center is visible only to authorized Admin or Super Admin users.

Use the Dashboard to find priority work; use individual records to make or verify changes. The navigation does not grant a permission by itself—roles and server-side policies remain authoritative.
