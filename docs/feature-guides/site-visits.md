# Site visits

Site visits record viewing appointments for a buyer and project/lot. Capture timing, location, assigned staff, notes, and outcome. Valid outcomes include scheduled, completed, no-show, cancelled, and rescheduled.

:::warning Site visit boundary
A visit does not reserve a lot, confirm buyer readiness, record a deposit, or create a payment.
:::

**Media:** Annotated screenshot, walkthrough video, checklist.
