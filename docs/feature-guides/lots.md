# Lots

Lots is the inventory view for parcel number, dimensions, size, zoning, base price, and core status: Available, Reserved, or Sold. Use it to verify current inventory before promising a lot to a buyer.

Active reservations are shown as separate readiness context. Confirm both the core lot record and active reservation context before acting.
