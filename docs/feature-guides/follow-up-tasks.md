# Follow-up tasks

Follow-up tasks create accountable sales reminders with a title, description, due date, priority, owner, and status. Active tasks should be completed, rescheduled, or cancelled with a reason—not silently ignored.

Open and in-progress tasks appear in dashboard and reporting queues. Overdue tasks require a documented next action or manager escalation.
