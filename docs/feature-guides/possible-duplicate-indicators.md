# Possible duplicate indicators

A possible duplicate indicator flags a record when lightweight matching finds shared email, phone, or name/lot-interest signals. It is a prompt for staff review.

:::warning It does not block or merge
The indicator does not prove two people are the same, block intake, or merge records. Compare the records and follow the approved data-correction process.
:::
