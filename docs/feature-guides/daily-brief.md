# Daily Brief

Authorized Admin and Super Admin users can generate a morning or custom-period brief. It summarizes applications, lots, payments, contracts, collections, alerts, and recommended actions, with an action center for carryover work.

The brief is built from a deterministic source packet. Source-backed alerts retain their attention type, source record, related record, generated status, deadline, source update time, and destination link. The brief is a snapshot, not a replacement for the live Dashboard.

When you open a saved item, the platform rechecks its source record. It can show **Current**, **Updated since brief**, **Resolved since brief**, or **Source unavailable**. Keep the historical text for context, but use the current state and linked record to decide what to do next.

:::note Advisory only
The brief and any AI narrative are read-only operational guidance. They cannot approve/decline applications, record payments, update balances, reserve lots, or send email automatically.
:::

For the shared rules and drill-down examples, see [Operational Attention and drill-downs](./operational-attention.md).
