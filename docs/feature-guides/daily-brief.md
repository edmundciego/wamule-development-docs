# Daily Brief

Authorized Admin and Super Admin users can generate a morning or custom-period brief. It summarizes applications, lots, payments, contracts, collections, alerts, and recommended actions, with an action center for carryover work.

:::note Advisory only
The brief and any AI narrative are read-only operational guidance. They cannot approve/decline applications, record payments, update balances, reserve lots, or send email automatically.
:::
