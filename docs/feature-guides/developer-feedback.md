# Developer feedback

Use **Send Feedback** in the protected application to report a Bug, Question, Feature Request, Data Issue, or Other issue. Select a realistic priority, describe what happened, and include safe reproduction information. The current page address is captured.

Feedback is saved and may queue a notification for manual processing in the Email Center; it is not an automatic support inbox or guaranteed immediate response channel.

**Media:** Annotated screenshot, walkthrough video, checklist.
