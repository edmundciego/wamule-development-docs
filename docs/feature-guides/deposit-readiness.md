# Deposit readiness

Deposit readiness records expectation, due date, proof-submitted state, confirmation date, and optional link to an existing payment. Use it to focus follow-up work.

:::danger Not financial reconciliation
Deposit readiness cannot create a payment, confirm a bank transfer automatically, generate a receipt, or replace the payment ledger. Record and verify the payment separately.
:::
