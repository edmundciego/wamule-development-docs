# Reservations

Reservations track a temporary operational hold against a lot and can link to a lead, application, customer, existing payment, or contract. Staff record expiry, expected deposit, assignee, notes, and status.

Reservations do not automatically update core parcel availability, create payments, or alter balances. Release, cancellation, and expiry need staff action and accountability notes.
